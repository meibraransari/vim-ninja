Welcome to VimNinja Sandbox!
============================

This is your Vim practice environment.
You have real Vim and Neovim installed here.

Practice files in this directory:
  01-modes.txt      - Normal and Insert mode
  02-movement.txt   - hjkl and word navigation
  03-editing.txt    - Delete, change, yank
  04-visual.txt     - Visual mode practice
  05-search.txt     - Search and replace
  06-advanced.txt   - Text objects and macros

Start with:
  vim 01-modes.txt

Or use Neovim:
  nvim 01-modes.txt

Use tmux to split windows:
  tmux
  Ctrl+a | (vertical split)
  Ctrl+a - (horizontal split)

Good luck, Ninja! ⚡
